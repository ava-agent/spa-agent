# 按摩 SPA 知识大全 - 移动应用设计规划

## 应用概述

一个全面的按摩与 SPA 知识百科应用，为用户提供专业、易懂的健康养生知识。采用静态展示模式，无需登录，用户可随时浏览各类知识内容。

---

## 屏幕列表

### 1. **首页 (Home)**
- **内容**: 知识分类卡片网格，快速导航到各个知识板块
- **功能**: 
  - 展示 5 个主要分类（按摩基础、技法、精油、穴位、SPA 礼仪）
  - 每个分类卡片显示图标、标题、简短描述
  - 点击卡片跳转到对应分类详情页

### 2. **分类详情页 (Category Detail)**
- **内容**: 该分类下的所有文章/知识点列表
- **功能**:
  - 显示分类标题和描述
  - 列出该分类下的所有知识项目
  - 每项显示标题、简短摘要、阅读时间估算
  - 点击列表项跳转到知识详情页

### 3. **知识详情页 (Knowledge Detail)**
- **内容**: 单个知识项目的完整内容
- **功能**:
  - 显示标题、发布日期、阅读时间
  - 展示详细的知识内容（富文本格式）
  - 包含相关图片/插图
  - 底部显示相关推荐链接

### 4. **搜索页 (Search)**
- **内容**: 全局搜索功能
- **功能**:
  - 搜索框输入关键词
  - 实时搜索结果展示
  - 结果按相关性排序
  - 点击结果跳转到知识详情页

### 5. **收藏页 (Favorites)** [可选，使用本地存储]
- **内容**: 用户收藏的知识项目
- **功能**:
  - 显示已收藏的知识列表
  - 支持取消收藏
  - 本地 AsyncStorage 持久化

## 关键用户流程

### 流程 1: 浏览知识
1. 用户打开应用进入首页
2. 首页展示 5 个分类卡片
3. 用户点击某个分类卡片（如"按摩技法"）
4. 进入分类详情页，显示该分类下的所有知识项目列表
5. 用户点击某个知识项目
6. 进入知识详情页，显示完整内容

### 流程 2: 搜索知识
1. 用户点击底部标签栏的"搜索"选项
2. 进入搜索页面
3. 用户输入关键词（如"薰衣草"）
4. 应用实时显示搜索结果
5. 用户点击某个结果
6. 进入知识详情页

### 流程 3: 收藏知识 [可选]
1. 用户在知识详情页点击"收藏"按钮
2. 知识被保存到本地收藏列表
3. 用户可在"收藏"标签页查看所有收藏项目
4. 用户可点击收藏项目快速访问

## 颜色方案

### 品牌色彩
- **主色 (Primary)**: `#8B7355` - 温暖的棕色，代表自然、放松、护理
- **辅助色 (Secondary)**: `#E8D5C4` - 柔和的米色，营造温暖舒适感
- **强调色 (Accent)**: `#D4A574` - 金棕色，用于重要按钮和交互元素

### 中性色
- **背景色**: `#FEFDFB` - 极浅米色背景
- **文本色**: `#2C2C2C` - 深灰色文本
- **边框色**: `#E0D5CC` - 浅棕色边框
- **成功色**: `#6BA587` - 柔和绿色
- **警告色**: `#D4A574` - 金棕色
- **错误色**: `#C85A54` - 柔和红色

### 深色模式
- **背景色**: `#1A1A18` - 深棕色背景
- **文本色**: `#F5F1ED` - 浅米色文本

## 设计原则

1. **简洁易用**: 清晰的导航结构，用户可快速找到所需知识
2. **视觉舒适**: 采用温暖、放松的色彩，营造 SPA 般的氛围
3. **可读性优先**: 充足的行高、字体大小，确保长文本易读
4. **一手操作**: 所有交互元素位于屏幕下半部分，支持单手操作
5. **响应式布局**: 适配不同屏幕尺寸（手机、平板）

---

## 技术实现细节

- **框架**: Expo + React Native + TypeScript
- **样式**: NativeWind (Tailwind CSS)
- **导航**: Expo Router
- **本地存储**: AsyncStorage（用于收藏功能）
- **搜索**: 客户端全文搜索（基于 JSON 数据）
- **图片**: 本地资源或网络 URL

interface KnowledgeItem {

  imageUrl?: string;

}
